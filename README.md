## Project Name : MedPlus - Simple & Secure Multi-Vendor Online Pharmacy
***MedPlus*** is an easy-to-use and secure online marketplace for buying and selling medicines. It is not just a regular online shop; it is a Multi-Vendor platform. This means different verified sellers can list their medicines, and customers can buy them directly from one place.
We built this platform with a focus on Security and Simplicity. To keep everything organized, we use a Role-Based system for three types of users: Customers, Sellers, and Admins. To keep your data safe, we use ***Better-Auth*** for secure Session Management. Also use ***resend*** for email verification This ensures that every user stays logged in safely and their information is protected. Whether you are a buyer or a seller, MediStore provides a trusted environment for everyone.
## Role-Based Access Control (RBAC)
### 🛒 What a Customer Can Do
* Easy Login: A user can register or log in directly using Google.
* Find Medicines: Search by name or browse by category.
* Cart & Buy: Add items to the cart and buy them easily.
* Flexible Payment: Use bKash, Nagad, or any Card.
* Track Order: Check the Dashboard for "Delivered" status.
* Review: Give a review after receiving the medicine.
## 👨‍⚕️ What a Seller Can Do
* Easy Login: A user can register to become a seller.
* Manage Inventory: A user can view their medicines and delete them if needed.
* Track Orders: See which customer ordered their specific medicines.
* Post Medicines: Upload new medicines with photos and details.
* Update Status: Change order status from "Shipping" to "Delivered."
* View Reviews: Check feedback and ratings from customers.
## 🛡️ What an Admin Can Do
* Manage Categories: A user with an admin role can create and manage medicine categories.
* Monitor Reviews: View and monitor all customer reviews for every medicine on the platform.
# 🛠️ Tech Stack</u>
### Frontend:
* Next.js: For building a fast and SEO-friendly user interface.
* TypeScript: For writing clean and type-safe code.
* Tailwind CSS: For modern and responsive styling.
* Shadcn UI: For beautiful and accessible UI components.
### Backend & Database:
* Node.js & Express: For building a powerful backend API.
* PostgreSQL: As a reliable relational database.
* Prisma: As an ORM to communicate easily with the database.
### Validation 
*Zod
### Authentication & Services:
* Better-Auth: For secure user authentication and session management.
* Resend: For sending professional emails (verification, orders).
### Payment: 
* SSLCommerz.
### Deployment:
* Vercel: For hosting the frontend (Next.js).
* Render: For hosting the backend server and database.
# ✨ Key Features
* User Roles: Separate Dashboards for Customers, Sellers, and Admins.
* Secure Login: Easy Google Login and session management with Better-Auth.
* Multi-Vendor: Different sellers can list and sell medicines in one place.
* Smart Search: Fast search by name and medicine Categories.
* Cart & Checkout: Simple shopping cart and secure payment (bKash, Nagad, Card).
* Order Tracking: Live status updates from "Shipping" to "Delivered."
* Review System: Customers can rate and review medicines after delivery.
* Email Alerts: Automated email notifications using Resend.
* Mobile Ready: Fully responsive design for all devices.
